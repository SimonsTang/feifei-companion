## 🌟 项目简介 | Project Overview

**飞飞学伴（FeiFei Companion）**是一套面向中国K12家庭的AI智能教育陪伴系统，采用**三位一体**架构设计：

```
┌─────────────────────────────────────────┐
│          🚀 飞飞学伴 v4.0               │
│       FeiFei Companion v4.0             │
│                                         │
│   🌸 飞飞老师 — 总管调度                 │
│   📚 小飞学姐 — 文科辅导                 │
│   🔬 浩云学长 — 理科辅导                 │
│   💞 亲子翻译官 — 家庭沟通               │
└─────────────────────────────────────────┘
```

**一个入口，全面覆盖K12家庭教育场景：**
- 🎓 **学科辅导** — 语文/数学/英语/物理/化学/编程
- 📚 **文科精讲** — 诗词赏析/作文批改/英语提升
- 🔬 **理科攻克** — 第一性原理/跨学科推导/编程启蒙
- 👨‍👩‍👧 **亲子沟通** — 翻译官/心理陪伴/家庭教育
- 🧠 **知识网络** — 全科知识图谱/薄弱点定位/智能推荐

---

## ✨ 核心特性 | Features

| 特性 | 描述 |
|------|------|
| 🌸 **三体合一** | 一个项目集成总管+文科+理科+亲子四大模块 |
| 🧠 **知识网络** | 全科知识图谱，智能定位薄弱点，前置知识自动检测 |
| 📚 **教材同步** | 紧扣人教版/统编版等主流教材，考点精讲 |
| 🎯 **Agent协调** | 问题自动分派给最适合的AI老师 |
| 💬 **亲子沟通** | 双向翻译+情绪疏导，化解家庭学习矛盾 |
| 📊 **数据驱动** | 掌握度追踪、学习报告、遗忘曲线复习提醒 |
| 🔄 **心跳任务** | 主动AI：主动唤醒、发任务、校验、循环 |
| 🎓 **名校背景** | UCL教育学/物理系学霸角色设定 |
